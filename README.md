Tijdens deze sprint "fix the flow interactive website" heb ik me verdiept in het ontwerpen en maken van gebruiksvriendelijke en toegankelijke interacties. Ik heb een hamburger menu gemaakt en een favorieten functie, en een scroll-animatie toegevoegd als interacties voor mijn website. Het hamburger menu werkt op mobiel en tablet devices, dit zorgt ervoor dat de navigatie-links niet te veel ruimte innemen op deze kleinere devices. De favorieten functionaliteit heb ik toegevoegd aan de evenementen. Door middel van deze functie kun je de evenementen die jou aanspreken bewaren door op het hartje te klikken. En tot slot heb ik een scroll-animatie toegevoegd aan de kopjes tekst onderin de pagina als mooie interactie, zodat de pagina niet te statisch lijkt.
