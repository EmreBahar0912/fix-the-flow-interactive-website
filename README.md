Tijdens deze sprint "fix the flow interactive website" heb ik me verdiept in het ontwerpen en maken van gebruiksvriendelijke en toegankelijke interacties. Ik heb een hamburger menu gemaakt en een favorieten functie, en een scroll-animatie toegevoegd als interacties voor mijn website. Het hamburger menu werkt op mobiel en tablet devices, dit zorgt ervoor dat de navigatie-links niet te veel ruimte innemen op deze kleinere devices. De favorieten functionaliteit heb ik toegevoegd aan de evenementen. Door middel van deze functie kun je de evenementen die jou aanspreken bewaren door op het hartje te klikken en komt erna een succesmelding tevoorschijn als feedback voor de gebruiker. En tot slot heb ik een scroll-animatie toegevoegd aan de kopjes tekst onderin de pagina als mooie interactie, zodat de pagina niet te statisch lijkt.

## Kenmerken
<!-- Bij Kenmerken staat welke technieken zijn gebruikt en hoe. Wat is de HTML structuur? Wat zijn de belangrijkste dingen in CSS? Wat is er met JS gedaan en hoe? -->
Ik heb de interacties gemaakt met JavaScript met behulp van het 3-stappenplan en de animaties gestyled met CSS. Ik heb gebruik gemaakt van een mobile first aanpak en dit vervolgens opgebouwd naar een groter viewport. De evenementen cards bestaat op mobiel uit 1 kolom, tablet 2 kolommen en laptop 3 kolommen. Ook heb ik voor mijn animaties van de interacties gebruik gemaakt van keyframes. Voor mij was de intersectionObserver het uitdagendste onderdeel deze sprint. Hiermee heb ik voor het eerst gewerkt om de scroll-animatie met een fade-in werkend te krijgen.
